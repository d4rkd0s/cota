What's here
-----------
One SVG per ISO 3166-1 alpha-2 country code needed by
bin/country_borders.json, named "<lowercase-iso2>.svg" (e.g. us.svg,
fi.svg). Files are copied verbatim from the flag-icons "4x3" flag set
(non-square aspect ratio flags at their natural proportions) and are
otherwise unmodified.

Why this source
----------------
flag-icons is MIT licensed, widely used, keyed by ISO-3166-1-alpha-2
filenames (a direct match for this project's iso2 codes), and ships
small, crisp, dependency-free SVGs suitable for baking into an
offline-first app. License text was verified directly against the
upstream repository (LICENSE file and GitHub API license metadata)
before bulk-downloading, per project policy of not trusting a cached
assumption about a license.
